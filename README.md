# 传送带弹簧振子模拟器

一个基于 Python 的交互式物理模拟程序，模拟传送带上连接弹簧的小方块运动。

## 物理模型

质量为 m 的小方块放置在匀速运动的传送带上，方块左侧通过弹簧连接至固定墙壁。传送带通过摩擦力驱动方块运动，弹簧提供恢复力，形成**粘滑振荡 (stick-slip oscillation)**。

### 核心方程

- **弹簧力**: F_spring = -k · (x - L₀)
- **静摩擦**: μ_s = ratio · μ_k，当 |F_spring| ≤ μ_s · m · g 时方块随传送带运动
- **动摩擦**: F_friction = -μ_k · m · g · sign(v - v_belt)

## 安装与运行

### 依赖

- Python ≥ 3.8
- numpy
- matplotlib

### 启动

```bash
python main.py
```

## 操作说明

### 控制面板 (右侧)

| 控件 | 作用 |
|------|------|
| **k 滑块** | 调节弹簧劲度系数 (1~40 N/m) |
| **μ_k 滑块** | 调节动摩擦系数 (0.01~0.8) |
| **μ_s/μ_k 滑块** | 调节静摩擦倍数 (1.0~2.5) |
| **v_belt 滑块** | 调节传送带速度 (0.1~2.0 m/s) |
| **m 滑块** | 调节方块质量 (0.1~5.0 kg) |
| **▶ (播放)** | 继续仿真 |
| **⏸ (暂停)** | 暂停仿真 |
| **↺ (重置)** | 重置到初始状态 |

### 界面

- **主场景**: 显示墙壁、弹簧(蓝色折线)、方块(紫色)、传送带(带运动条纹)
- **力箭头**: 在方块上方显示 F_spring(红)、F_friction(蓝)、F_net(绿)；右侧显示重力(橙)和支持力(金)
- **X-T 图**: 左下角实时绘制位置-时间曲线，自动缩放
- **状态指示**: 方块上方显示 "粘滞" 或 "滑动" 状态和实时速度
